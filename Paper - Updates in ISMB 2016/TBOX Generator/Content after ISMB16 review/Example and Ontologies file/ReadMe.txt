This folder contains files related to ontology updates required by reviewers from ISMB 2016.
There is a btl2.owl file to support ontology developtment.
There is a example.owl, used to derive the examples for VBA data propagation in a new OWL file.

GO, ChEBI and PRO are compressed as all of them have more than 100mb. Github does not accept files with more than 100mb.
